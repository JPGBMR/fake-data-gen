# Fake Data Generator

[![Live Demo](https://img.shields.io/badge/Live%20Demo-jpgbmr.github.io-58a6ff?style=flat-square)](https://jpgbmr.github.io/fake-data-gen/)
[![HTML](https://img.shields.io/badge/HTML5-E34F26?style=flat-square&logo=html5&logoColor=white)](https://github.com/JPGBMR/fake-data-gen)

Generate realistic fake data for forms, databases, and demos. Names, emails, phones, addresses, UUIDs, credit cards (Luhn-valid test numbers), dates, and more. Export as JSON, CSV, or plain list.

## Getting Started

Open `index.html` in your browser. No install required.
